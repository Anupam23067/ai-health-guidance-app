# AI Health Guidance App

This project is an AI-powered health guidance application developed using FlutterFlow.

## Features
- Symptom input
- Disease image upload
- AI-based health guidance
- Emergency level indication
- Color-coded health status
- Doctor guidance (demo data)

## Technologies Used
- FlutterFlow
- Flutter
- Firebase
- OpenAI API

## MVP
The MVP demonstrates user symptom input and AI-based health analysis.
